# Technology Stack

## Build System & Runtime

- **Build Tool**: Vite 5.0.0
- **Runtime**: Node.js 18+
- **Package Manager**: npm
- **Language**: TypeScript 5.3.0
- **Module System**: ES modules

## Frontend Framework

- **React**: 18.2.0
- **React DOM**: 18.2.0
- **JSX**: React JSX (configured in tsconfig.json)

## AWS Services & SDKs

- **@aws-sdk/client-s3**: ^3.500.0 - S3 operations (list, upload, download, delete)
- **@aws-sdk/s3-request-presigner**: ^3.500.0 - Signed URL generation
- **@aws-sdk/client-cognito-identity**: ^3.500.0 - Cognito Identity Pool client
- **@aws-sdk/credential-provider-cognito-identity**: ^3.500.0 - Credential provider for Cognito

## Development Tools

- **@vitejs/plugin-react**: ^4.2.0 - React support in Vite
- **@vitejs/plugin-basic-ssl**: ^1.0.0 - HTTPS support for local development
- **@types/react**: ^18.2.0 - React type definitions
- **@types/react-dom**: ^18.2.0 - React DOM type definitions
- **@types/node**: ^20.0.0 - Node.js type definitions

## TypeScript Configuration

- **Target**: ES2020
- **Module**: ESNext
- **Strict Mode**: Enabled
- **JSX**: React
- **Module Resolution**: Bundler
- **Unused Variables/Parameters**: Errors on unused code

## Common Commands

### Development
```bash
npm run dev
```
Starts Vite dev server on `http://localhost:8080` with HTTPS and auto-open browser.

### Build
```bash
npm run build
```
Runs TypeScript compiler check, then builds optimized production bundle to `dist/` folder.

### Preview
```bash
npm run preview
```
Serves the production build locally for testing.

### Installation
```bash
npm install
```
Installs all dependencies from package-lock.json.

## Environment Variables

Development uses `.env.local` (not committed):
- `VITE_AWS_REGION` - AWS region (e.g., us-east-1)
- `VITE_COGNITO_IDENTITY_POOL_ID` - Cognito Identity Pool ID
- `VITE_S3_BUCKET_NAME` - S3 bucket name

Production uses `public/config.json` (loaded at runtime).

## Code Style & Standards

- **Strict TypeScript**: All code must pass strict type checking
- **No Unused Code**: Compiler enforces no unused variables or parameters
- **Functional Components**: React components use functional component pattern with hooks
- **Service Layer Pattern**: Business logic separated into services (authService, s3Service)
- **Type Safety**: All interfaces defined in `src/types/index.ts`

## Build Output

- **Output Directory**: `dist/`
- **Contents**: Optimized production bundle with compiled JavaScript, CSS, and static assets
- **Generated by**: `npm run build` command
- **Ready for**: Static hosting (CDN, S3, web servers, etc.)
- **Note**: The `dist/` folder is generated and not committed to version control

## Browser Support

- Modern browsers with ES2020 support
- HTTPS required (enforced in dev via basicSsl plugin)
